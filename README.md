# Browser JSON Schema Validator Demo

This example demonstrates a browser-based JSON Schema validator using the AJV library. It showcases validation against a JSON Schema (Draft-07) including features like $ref, allOf, anyOf, and if/then/else, with results displayed directly in the browser.

## Language

`html`

## How to Run

Open the `index.html` file in your web browser. Enter your JSON Schema and JSON data into the text areas and click 'Doğrula' (Validate).
